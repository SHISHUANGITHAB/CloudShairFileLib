# 项目审批云端共享档案库平台

基于 Vue + Java（Spring Boot）的档案管理平台，实现政府端与企业端之间的项目审批档案数字化管理。

## 功能模块

### 政府端
- **首页** — 推送消息、待办事项、数据总览（档案统计/在线人数/操作动态）
- **档案管理** — 档案查询（多条件组合检索/全文检索/标签筛选）、档案新增/上传、权限控制
- **系统管理** — 统一登录集成、用户/角色权限管理、档案分类/标签配置
- **日志管理** — 操作日志查询与导出

### 企业端
- **首页** — 档案统计、快捷入口、消息通知
- **档案库** — 自主上传、档案列表查看、在线预览、单文件/批量下载
- **日志查看** — 企业操作记录

## 原型系统

`protosystem/index.html` 为可运行的 HTML 原型（Tailwind CSS + FontAwesome），支持政府端/企业端角色切换，含 Mock 数据。
